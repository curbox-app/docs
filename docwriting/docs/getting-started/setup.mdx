Curbox needs a few special permissions to do its job. This page walks you through each one. The whole process takes about five minutes.


## Install the App

Download the latest `.apk` file from the Curbox releases page and open it to install. Android may ask you to allow installs from unknown sources — this is normal for apps outside the Play Store.


## Grant Permissions

When you first open Curbox, an onboarding screen guides you through five permissions. You can grant them one by one, or use Shizuku to grant them all automatically.

<Aside type="tip">
Shizuku is a free helper app that can grant all five permissions with one tap. If that sounds appealing, install Shizuku before opening Curbox for the first time.
</Aside>

<Steps>
1. **Screen Overlay**
   This lets Curbox display a pause screen on top of other apps when a block triggers. Tap the button, find Curbox in the list, and turn on **Display over other apps**.

2. **Notifications**
   Curbox uses a small notification to keep its service running in the background. Tap the button and allow notifications.

3. **Do Not Disturb**
   Required if you want Curbox to turn on Do Not Disturb automatically. Tap the button and grant access.

4. **App Blocker Accessibility**
   This is the main permission. It lets Curbox see which app is on screen so it can block it when needed. Tap the button, find **Curbox** in the list, and turn it on.

5. **Usage Tracker Accessibility**
   This lets Curbox track how long you spend in each app. Tap the button, find **Curbox** in the list, and turn it on.
</Steps>

<Aside type="note">
Accessibility Services sound powerful, and they are. Curbox only uses them to detect the foreground app and measure your usage time. It never reads your messages, passwords, or anything else you type.
</Aside>


## You Are Ready

Once all five permissions are granted, Curbox is active. Head to the [Reducers](BASE_URL/reducers/overview) tab to set up your first limit.

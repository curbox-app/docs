## What is a Focused Session?

A focused session is a time-boxed period during which you designate certain apps as *blocked*. While the session is active, opening those apps will automatically kick you out of them to reinforce focus.

Sessions can be confiured to automatically run on specific time period each day of the week,
or manually turn on whenever you require

- **Auto Focus** — You start them manually when you need for a fixed duration
- **Manual Focus** — You specify in advance between what time periods you'd like the focus session to run

## Starting a Session

<Steps>

1. **Open Curbox** and tap the focus button on the bottom.
   -- Screenshot of the focus screen --

2. **Choose a duration**

   Use the slider to pick a duration. Common presets (15 min, 30 min, 160 min, 120 min) appear as quick-select chips.
   We recommend you to set a focus duration of 2-3 mins for initial test.

   <Aside type="tip">
   Research on sustained attention suggests 90 minutes as the natural limit of a deep work block. For beginners, 25-minute Pomodoro sessions are a great starting point.
   </Aside>

3. **Concept of groups**

   Curbox entirely is based on the concept of App groups. You make configurations in advance which you can quickly run later. This is 
   When you press Start Focus button at the bottom, you'll be presented with a dialog with an option to select focus group.
   Press the "Create New Group" button.

4. **Creating a new focus group**

   -- Screenshot of creating focus group --

   Here you need to enter the following information
   Group name - Name of the focus group. This also acts as a tags on analytics screen
   A select apps button that lets you select apps you want to run focus mode on

   Now below that you'll have an option called "let me quit mid sessions"
   if you uncheck this YOU WONT BE ABLE TO QUIT FOCUS MODE ONCE IT STARTS UNTIL IT ENDS
   
   Click save group

5. **Starting focus mode**
   Select the group you just created and press start focus. 
   The focus mode has been turned on!
   
   Curbox minimises itself. A persistent notification shows the remaining time. 


</Steps>
